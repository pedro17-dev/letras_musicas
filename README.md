# 🎵 Letras Musicais - Streamlit App

Aplicação web desenvolvida em **Python utilizando Streamlit**, que permite pesquisar letras de músicas através da integração com uma **API externa**.

O projeto tem como objetivo demonstrar o consumo de APIs, criação de interfaces interativas e desenvolvimento de aplicações web utilizando Python.

---

## 📌 Funcionalidades

✅ Busca de letras de músicas  
✅ Integração com API externa  
✅ Interface web interativa utilizando Streamlit  
✅ Tratamento de erros nas requisições  
✅ Exibição dinâmica dos resultados  
✅ Aplicação responsiva e simples de utilizar
